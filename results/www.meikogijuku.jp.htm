  • いつから入会できますか？
    • Answer :

    明光義塾は個別指導なので小・中・高の全ての学年で、いつからでも授業をスタートすることができます。現在のお子さまの学力や目標に合わせて最適な学習プランを提案いたします。まずはお近くの教室へお気軽にご相談ください。

    Question :
  • 週何回ぐらい通うのが良いのですか？
    • Answer :

    最適な通塾回数は一人ひとり異なります。明光義塾では、お子さまの学力や今後の目標やご要望、診断テストの結果などをふまえて、一人ひとりに最適な通塾回数をご提案いたします。まずはお近くの教室へお気軽にご相談ください。

    Question :
  • クラスに分かれているのですか？
    • Answer :

    クラス分けやクラス単位での授業は行っていません。お子さま一人ひとりに合わせた学習プランを作成し、それに基づいて個別指導を行っていきます。現在の学力に応じたところからスタートしますので、安心してご入会いただけます。まずはお近くの教室へお気軽にご相談ください。

    Question :
  • 個別指導は家庭教師と比べて何が良いのですか？
    • Answer :

    家庭教師は「自分だけ」の先生ですが、自分自身で「考える」よりも前に「教えてもらう」という依存心が生じてしまいがちです。
    個別指導では、講師1人が生徒数人を受け持ち、指導する際にはマンツーマンになります。明光義塾では自分自身の力で考え、答えを導き出せるような指導を行いますので、自立心を育てながら学習効果を上げることができます。また、教室長はお子さまの学習状況をしっかりと管理していますので、的確な進路指導を行うことができます。

    Question :
  • どんな教科でも受講できますか？
    • Answer :

    はい、できます。明光義塾は全学年・全教科に対応していますので、「苦手な教科を克服したい」「得意な教科をもっと伸ばしたい」など、ご希望に応じて受講する教科を選ぶことができます。まずはお近くの教室へお気軽にご相談ください。

    Question :
  • 宿題は出ますか？
    • Answer :

    はい、出ます。宿題は授業の内容を定着させ、家庭学習を習慣づけるために必要不可欠です。お子さまの学習状況に応じて、適切な量の宿題をお出ししています。

    Question :
  • 途中で教科を変更したり、受講教科を増やしたりできますか？
    • Answer :

    はい、できます。お子さまの学習状況により、受講している教科を別の教科に変更したり、受講教科を追加することが可能です。教室長へお気軽にご相談ください。

    Question :
  • 授業はどのように進めるのですか？
    • Answer :

    「講師がポイントを教える」→「生徒自身が問題を解いてみる」→｢講師と一緒に理解度をチェックする」というサイクルを繰り返しながら、生徒が自分の力で答えを導き出せるよう授業を進めていきます。明光義塾の授業について、さらに知りたい方はお近くの教室へお気軽にご相談ください。

    Question :
  • 入会するためのテストはありますか？
    • Answer :

    明光義塾では入会の可否のためのテストは一切行っておりません。入会に際して「明光診断テスト」を行うことがありますが、最適な学習プランの作成や教材選定の際の参考にさせていただくものとなります。お子さまの学力や目標に合わせて最適な学習プランをご提案しますので、安心してご入会いただけます。まずはお近くの教室へお気軽にお問い合わせください。

    Question :
  • お休みした場合の授業はどうなるのですか？
    • Answer :

    教室で定めた期日までに連絡があった欠席について、特別な事情があると認められたときは、授業を他の日へ変更することができます。ただし原則として同じ月の空席のある時間に限りますので、教室長へお気軽にご相談ください。

    Question :
  • 前学年の内容を復習することはできますか？
    • Answer :

    はい、できます。明光義塾は「どこからわからなくなったのか」をつきとめ、必要であれば学年をさかのぼって指導します。まずはお近くの教室へお気軽にご相談ください。
